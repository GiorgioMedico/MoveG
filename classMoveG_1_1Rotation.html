Detailed Description

Classe per rappresentare e gestire rotazioni nello spazio tridimensionale.

Questa classe supporta diverse rappresentazioni di rotazioni, inclusi matrici di rotazione, quaternioni, asse-angolo e angoli di Eulero. Permette inoltre la composizione di rotazioni.

Rotation MoveG::Rotation::fromRotationMatrix ( const Eigen::Matrix3d &  rotation_matrix)
static

Crea una rotazione da una matrice di rotazione.

Metodi statiti per definire una rotazione I metodi statici permettono di creare una rotazione senza dover istanziare un oggetto.

Esempio: Rotation rot = Rotation::fromEulerAngles(0, 0, 0, true, "XYZ", false); Altrimeti si sarebbe dovuto fare:

Rotation rot; rot = rot.fromEulerAngles(0, 0, 0, true, "XYZ", false);
